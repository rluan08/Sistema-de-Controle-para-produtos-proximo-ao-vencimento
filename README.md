# Sistema de Controle de Validade 
Aplicação web para registrar e organizar produtos que estão proximos ao vencimento.
## Funcionalidade
- Cadastro de Produtos
- Lista dos Produtos
- Opção de Editar/Excluir 
- Geração de Relatorio PDF
## Teclogias
- Html
- Css
- JavaScript
## Como executar
1. Baixe o repositorio.
2. Abra o arquivo index.html no navegador.
